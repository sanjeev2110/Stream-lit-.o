Steps for Hardware Integration

  1. Mount the edge device into the machine cabinet.
  2. Supply 24V power to the edge device.
  3. Connect shopfloor ethernet for the edge device to the central access switch.
  4. Obtain the IP address for the edge device from the local IT team.
  5. Configure the IP address for ETH1 on the edge device.
  6. Set up the edge device in the DATATRONiQ dashboard.
  7. Connect edge device with the PLC (e.g., Siemens S7-300/400, or else).
  8. Follow the PLC type specific integration steps that are available within the Equipment Integration Guide.
  9. Incorporate the edge device into the DATATRONiQ server cluster.
Attention: Right now there is no approved small edge device available, please go with the large edge device or with a low frequency data collection method.
